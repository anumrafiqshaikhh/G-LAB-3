# G-LAB 3: Interactive Button

## Description

This project demonstrates a simple interactive button using HTML, CSS, and JavaScript. The button responds to user clicks by updating the status message and limiting the number of interactions.

## Features

* Button click interaction
* Dynamic status updates
* Click limit with disable functionality
* Basic visual feedback (animation & styling)

## How it Works

* Each click updates the status message
* After a fixed number of clicks, the button gets disabled
* Visual feedback is shown for user actions

## Purpose

To understand basic user interaction and event handling in web development.
